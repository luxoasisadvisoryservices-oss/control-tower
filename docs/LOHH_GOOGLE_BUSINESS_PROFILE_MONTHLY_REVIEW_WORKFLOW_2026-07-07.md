# Lux Oasis Google Business Profile Monthly Review Workflow

Date: 2026-07-07
Business: Lux Oasis Holiday Homes / Oasis 2 presence
Cadence: Monthly, 5th day of each month at 10:00 Asia/Dubai
Cron: `4eb2dc9d-52b5-4bdd-996e-a3d92a267073`

## Purpose

Use the forwarded Google Business Profile performance report as a recurring growth input, not routine email noise.

The monthly check should identify what action is needed to increase:

- Google Business Profile interactions
- website clicks
- calls
- direction requests
- search visibility
- photo/content performance
- customer interaction
- direct-booking intent from Google to the website
- SEO signals that support more direct enquiries

## Monthly Check

1. Search the business Gmail for the latest forwarded Google Business Profile / Business Profile Performance report.
2. Extract the visible metrics from the report.
3. Compare against the previous saved report if available.
4. Identify what moved up, what moved down and what is flat.
5. Produce a short action list for:
   - Google Business Profile updates
   - website SEO
   - direct-booking conversion
   - photos/posts/content
   - review/customer interaction
   - tracking gaps
6. Update Control Tower and memory with the monthly result.

## Strategy

Use Google Business Profile as the local discovery and trust layer. Use the website as the direct-booking conversion layer.

The monthly strategy is:

1. Increase discovery: make more people find Lux Oasis through Google Search and Maps.
2. Increase trust: keep photos, services, reviews and updates fresh enough that visitors believe the business is real and active.
3. Increase action: push more users to call, click the website, request directions or contact through the booking path.
4. Increase conversion: make sure the website page they land on has clear property/service copy and a strong direct-booking or enquiry CTA.
5. Learn from the numbers: each month, pick only the 3-5 highest-impact actions instead of creating a long generic SEO list.

## Monthly Action Framework

When the report arrives, classify the month into one main state:

- Visibility problem: views/searches are flat or falling.
- Conversion problem: views are decent, but calls/clicks/interactions are weak.
- Trust problem: profile has stale photos, weak review activity or no recent posts.
- Website problem: website clicks happen, but direct enquiry or booking is not improving.
- Tracking problem: the report does not show enough data to know what is working.

Then produce a short action list:

1. Profile action: what to update on Google Business Profile.
2. Content action: what post, photo, service update or offer to prepare.
3. Website action: what page, copy, CTA or SEO improvement to make.
4. Review action: what guest/customer review or reply opportunity exists.
5. Tracking action: what UTM, booking link or analytics check is missing.

## Step-By-Step Operating Procedure

1. Open the latest forwarded report email.
2. Save the reporting period and source email date.
3. Record the core metrics:
   - profile views
   - searches
   - calls
   - website clicks
   - directions
   - messages or bookings if visible
   - photo views if visible
4. Compare with the previous month.
5. Pick the main issue: visibility, conversion, trust, website or tracking.
6. Draft 3-5 actions only.
7. Separate internal actions from approval-needed actions.
8. Update Control Tower with the result.
9. If action needs approval, send Damiano a short approval card.

## Example Output Format

- GBP result: calls up, website clicks down, directions flat.
- Diagnosis: conversion/website issue, not visibility issue.
- Action 1: prepare one Google post about direct booking / Oasis2 availability.
- Action 2: refresh Google photos with 5-8 better apartment/building/lifestyle images.
- Action 3: improve website CTA above the fold for direct booking/enquiry.
- Action 4: add UTM tracking to the Google website link.
- Approval needed: publish Google post / change profile link / edit live website.

## Approval Boundaries

Allowed autonomously:

- read the report
- summarise metrics
- compare trends
- prepare action recommendations
- draft Google post copy, website copy, SEO edits or content briefs
- update internal trackers and Control Tower

Needs Damiano approval:

- publishing Google Business Profile posts
- editing public business profile details
- changing website live copy
- launching ads or spending budget
- replying publicly to reviews
- making owner/client-facing promises
